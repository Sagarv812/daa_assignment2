# Test Cases

- `basic_square.in`: convex polygon, expected `2` triangles.
- `concave_gallery.in`: simple concave polygon, expected `6` triangles.
- `one_hole_square.in`: one hole inside a square, expected `8` triangles.
- `two_holes_rect.in`: two rectangular holes, expected `14` triangles.
- `irregular_outer_irregular_hole.in`: irregular outer polygon with a pentagonal hole.
- `concave_outer_triangle_hole.in`: concave outer polygon with a triangular hole.
- `irregular_two_holes.in`: irregular outer polygon with two non-square holes.
- `collinear_two_holes.in`: comb-shaped outer polygon with two octagonal holes and long collinear bridge opportunities, expected `42` triangles.
- `mixed_suite.in`: bundle of the original regression cases for one-shot testing.
- `hard_suite.in`: six deterministic stress cases generated by `generate_hard_suite.py`, covering crenellated corridors, aligned-hole ladders, dense multi-hole galleries, and non-axis-aligned star-shaped outers.
- `edge_case_suite.in`: six additional deterministic edge-case stress tests generated by `generate_edge_case_suite.py`, focused on same-wall bridge cascades, equal-y events, horizontal ledges, dense collinearity, and thin-channel mazes.
- `rigorous_suite.in`: ten larger deterministic stress cases generated by `generate_rigorous_suite.py`, combining wide equal-y bands, deep orthogonal mazes, dense convex grid packs, rotated stars, and repeated pocket clusters.
